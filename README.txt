Find Compatible WordPress / Plugin / PHP Version

== Description ==

Plugin will help you to Detect Specific Compatible Plugin Version for your Server PHP version and WordPress version via website <a href="https://wpcompatible.com/">wpcompatible.com</a>

Providing very simple way to solve compatibility issues in case for example when you stuck with older WordPress, older PHP version or similar scenario.

Plugin automatically detect your WP version and PHP version

You will enter wanted plugin name or select from currently installed

Plugin will help you to detect latest plugin version compatible with your server

== Installation ==

1. Extract `plugin-compatibility.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Can it detect any compatible plugin  =

No, our database is limited to most popular plugins

= How to use? =

In Dashboard go to Tools->Plugin Compatibility
